# Yeelight Controller
Python script to control Yeelight Bulbs.

## Install
Install controller:
```shell
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run
Make sure that your have developer mode enable on Yeelight app.
```shell
python main.py
```
